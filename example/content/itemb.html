How does it work?

jsTemplar uses a template-view design pattern modified for use with just Javascript. The aim of this design pattern is to create a theme, split it up based on semantic HTML5, and allow for rapid content addition without the need of any server side code. Tags that we recommend building around are:

      Header
      Nav
      Section
      Aside
      Footer

What is this pattern good for?

This design pattern is best suited static pages; such as:

      Landing pages
      Portfolios
      General information
      Company websites
      Documentation
      ...and more!
